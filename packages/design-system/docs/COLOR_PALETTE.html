Blue Palette 01

AK Golf Color System

Production-ready fargepallett for AK Golf Academy

Regel:
UI-komponenter skal ALDRI bruke rå farger (hex) direkte.
Kun semantiske tokens er tillatt i produktkode (f.eks. --bg, --card, --accent, --success).
Brudd på dette regnes som feil implementasjon og skal stoppes i code review.
Brand Colors (Raw palette / Brand layer)

Dette er rå fargeverdier (brand layer).
Disse skal ALDRI brukes direkte i UI-komponenter.
Alle komponenter skal bruke semantiske tokens (f.eks. --accent, --bg, --success).
Direkte bruk av hex-verdier i komponenter regnes som feil implementasjon.

Tillatt bruk: CSS-variabler, theming, dokumentasjon.
Forbudt bruk: Direkte i JSX/TSX, inline styles, komponenter.
