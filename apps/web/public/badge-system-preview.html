Monochrome-First Design Process

Phase 1: Pure Monochrome

Single color on dark background. Symbol must be instantly recognizable.

Phase 2: Add Rim Color

Tier color on rim only. Symbol stays monochrome.

Phase 3: Subtle Tint

Field gets 15% max color tint. Never full color fills.

7

Do / Don't

Do

  • Use consistent 1.5px stroke weight
  • Keep badges to 3-5 visual elements
  • Design in monochrome first
  • Apply color only to rim/accents
  • Test at 24px for legibility
  • Use geometric primitives
  • Leave 30%+ negative space

Don't

  • Mix multiple stroke weights
  • Use emoji as badge symbols
  • Add gradients or shadows
  • Fill symbols with solid color
  • Create detail that breaks at small sizes
  • Use arbitrary angles (37°, 52°)
  • Add decorative elements without meaning
