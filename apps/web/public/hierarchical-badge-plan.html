Hierarkisk Badge System

AK Golf Academy - Kategori-skalert belonningssystem

Konsept

Badge-krav skaleres basert pa spillerens kategori (A-K). En K-spiller (nybegynner) trenger bare 30% av kravet til en A-spiller (elite/tour-aspirant) for a oppna samme badge. Dette sikrer motiverende og oppnaelige mal for alle spillere.

Skaleringsregler

ABSOLUTTE (Skaleres)

  • totalSessions (antall okter)
  • totalHours (treningstimer)
  • currentStreak (dager pa rad)
  • perfectWeeks (perfekte uker)
  • earlyMorningSessions (for kl 09)
  • eveningSessions (etter kl 19)
  • resolvedBreakingPoints
  • tournamentPreps

RELATIVE (Skaleres IKKE)

  • completionRate (% fullfort)
  • speedImprovement (+X mph)
  • improvementFromLast (% bedre)
  • avgQuality (gjennomsnitt)
  • passed (bestatt test)
  • categoryBenchmark (kategori-mal)

KATEGORI-SPESIFIKKE

  • WAGR-rank badges (kun A-B)
  • Tour-card (kun A)
  • College-recruit (kun A-C)
  • Junior Champion (D+ under 18)
  • First Birdie (alle kategorier)

Implementering

// badge-calculator.ts - Oppdatert med kategori-skalering const CATEGORY_MULTIPLIERS = { 'A': 1.00, 'B': 0.95, 'C': 0.90, 'D': 0.85, 'E': 0.80, 'F': 0.70, 'G': 0.60, 'H': 0.50, 'I': 0.40, 'J': 0.35, 'K': 0.30 }; function getScaledRequirement( baseRequirement: number, playerCategory: string, scalingType: 'absolute' | 'relative' | 'none' ): number { if (scalingType !== 'absolute') return baseRequirement; const mult = CATEGORY_MULTIPLIERS[playerCategory] || 1.0; return Math.round(baseRequirement * mult); } // Eksempel: Training Warrior Level 4 getScaledRequirement(100, 'A', 'absolute'); // => 100 okter getScaledRequirement(100, 'K', 'absolute'); // => 30 okter // Eksempel: Reliable (completion rate) - skaleres IKKE getScaledRequirement(90, 'K', 'relative'); // => 90% (uendret)
