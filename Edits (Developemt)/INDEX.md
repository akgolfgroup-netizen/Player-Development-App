# 00. Inbox

Midlertidig mottak for nye filer og ideer som skal sorteres.

## Formål
- Samleplass for alt nytt som kommer inn
- Prosesser regelmessig (daglig/ukentlig)
- Flytt til riktig PARA-kategori etter vurdering

## Arbeidsflyt
1. Nye filer havner her
2. Vurder: Er dette et prosjekt, ansvarsområde, ressurs eller arkivverdig?
3. Flytt til riktig mappe
4. Hold Inbox tom

## Hvor skal ting?
| Type | Destinasjon |
|------|-------------|
| Aktivt prosjekt | `01. Projects/Active/` |
| Løpende ansvar | `02. Areas/` |
| Referansemateriale | `03. Resources/` |
| Ferdig/inaktivt | `04. Archive/` |
