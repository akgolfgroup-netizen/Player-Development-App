## Coach-intervjuer: Temaer og Spørsmål

### Tema 1: Periodisering

**Overordnet:**
- Hvordan periodiserer dere et golfår?
- Er E/G/S/T (Evaluering/Grunnperiode/Spesialiseringsperiode/Turneringsperiode) gjenkjennelig?
- Hvilke perioder brukte Viktor/Kristoffer?
- Hvor lang er typisk hver periode?

**Spesifikke spørsmål:**
```
□ Hva kjennetegner en god off-season periode?
□ Hvordan trapper man opp mot turneringssesong?
□ Hvordan balanseres teknikk vs spill i ulike perioder?
□ Hva gjør man i uka før en viktig turnering?
□ Hvordan ser recovery-perioden ut etter sesong?
```

### Tema 2: Ukentlig Struktur

**Overordnet:**
- Hvordan ser en typisk treningsuke ut?
- Hvordan varierer dette med spillernivå?

**Spesifikke spørsmål:**
```
□ Hvor mange timer/uke trener en Elite junior?
□ Hvor mange timer/uke trener en Tour-spiller?
□ Hvordan fordeles tid mellom:
  - Tee Total (driver, jern)
  - Innspill 200+
  - Innspill 150-200 m
  - Innspill 100-150 m 
  - Innspill 50-100 m 
  - Nærspill(Pitch,Chip,Lob,Bunker)
  - Putting
  - Fysisk trening
  - Mental trening
  - Spill på bane
□ Hvilke dager er typisk hvile?
□ Hvordan varierer intensitet gjennom uka?
```

### Tema 3: Treningsøkter

**Overordnet:**
- Hvordan struktureres en enkelt treningsøkt?
- Hva er typisk varighet?

**Spesifikke spørsmål:**
```
□ Hvordan ser oppvarming ut?
□ Hvor lang er en typisk teknikkøkt?
□ Hvor lang er en typisk spilløkt?
□ Hvordan avsluttes økter?
□ Hva er forskjell på "lett" vs "hard" økt?
□ Hvordan kombineres teknikk og spill i samme økt?
```

### Tema 4: Domener og Prioritering

**Overordnet:**
- Hvordan prioriteres ulike deler av spillet?
- Hva fokuserte Viktor/Kristoffer mest på?

**Spesifikke spørsmål:**
```
□ Er TEE/APPROACH/ARG/PUTT/PHYS en god inndeling?
□ Mangler noen kategorier? (Mental? Taktisk?)
□ Hvordan identifiseres svakheter?
□ Hvor mye tid brukes på svakheter vs styrker?
□ Hvordan måles fremgang i ulike domener?
```

### Tema 5: Nivåtilpasning

**Overordnet:**
- Hvordan tilpasses trening til spillernivå?
- Hva er forskjellen mellom nybegynner og elite?

**Spesifikke spørsmål:**
```
□ Hva fokuserer en HCP 20-spiller på?
□ Hva fokuserer en HCP 5-spiller på?
□ Hva fokuserer en +2 spiller på?
□ Når er man "god nok" på en ferdighet til å gå videre?
□ Hvordan progresjon fra junior til elite?
```

### Tema 6: Viktor Hovland Case Study

**Spesifikke spørsmål:**
```
□ Hvordan trente Viktor som junior (15-18 år)?
□ Hva var hans ukentlige volum?
□ Hvilke områder fokuserte han mest på?
□ Hvordan utviklet treningsmengden seg over tid?
□ Hva gjorde han annerledes enn andre?
□ Hvilke "breaking points" jobbet han med?
```

## Spillertesting: Protokoll

### Før Testing
```
For hver spiller, samle inn:
□ Navn/alder
□ Handicap/kategori
□ Ukentlig treningstid tilgjengelig
□ Mål (kortsiktig/langsiktig)
□ Selvopplevde styrker/svakheter
```

### Under Testing
```
□ Kjør gjennom Team Norway testbatteri (hvis mulig)
□ Observer treningsøkt
□ Vis generert treningsplan fra systemet
□ Få tilbakemelding:
  - Er dette realistisk?
  - Mangler noe?
  - Er progresjon fornuftig?
```

### Etter Testing
```
□ Dokumenter funn
□ Sammenlign systemets output med coach-anbefaling
□ Identifiser gap
```
