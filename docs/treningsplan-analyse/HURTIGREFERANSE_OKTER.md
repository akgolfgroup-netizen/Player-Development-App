## COACHING CUES - HURTIGREFERANSE

**Driver:**
- "Sweep, ikke slå ned"
- "3-2-1-FIRE for tempo"
- "Tee høyde = trajectory"

**Jern:**
- "Ball først, bakke etterpå"
- "Vekt fremover ved impact"
- "Flat venstre håndledd"

**Wedge:**
- "Svinglengde = avstand"
- "Tempo alltid det samme"
- "Stol på loften"

**Short Game:**
- "Se landingsspot først"
- "Commitment til valget"
- "Akselerér gjennom"

**Bunker:**
- "Slå sanden, ikke ballen"
- "Splash teknikk"
- "Åpen face, åpen stance"

**Putting:**
- "Les først, så commit"
- "Stroke lengde = avstand"
- "Putter face square"

## VANLIGE FEIL & FIX

| Problem | Årsak | Fix |
|---------|-------|-----|
| Topper driver | For bratt | Sweep-følelse |
| Fetter jern | Ball for langt frem | Ball midt stance |
| Chunks wedge | Decel | Akselerér! |
| Push putts | Face åpen | Gate drill |
