## Quick Reference

### Primary Colors
| Token | Hex | Usage |
|-------|-----|-------|
| Brand Primary | `#1A3D2E` | Headers, primary actions |
| Brand Gold | `#C9A227` | Accents, achievements |
| Surface | `#FAFAF9` | Backgrounds |
| Text Primary | `#1C1917` | Body text |

### Logo Clearspace
Minimum clearspace = 1x logo height on all sides

### Typography
- **Display:** SF Pro Display / Inter
- **Body:** SF Pro Text / Inter
- **Mono:** SF Mono / JetBrains Mono
