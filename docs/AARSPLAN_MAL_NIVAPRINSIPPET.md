## 5. BREAKING POINT INTEGRASJON

### Identifisering (basert på testresultater)

```
Breaking Point = Test hvor resultat < Kategori-krav

Eksempel Spiller D:
✅ Driver: 245m (krav: 240m)
❌ Wedge PEI: 0.085 (krav: 0.07) → BREAKING POINT
✅ Putting 3m: 78% (krav: 75%)
❌ Bunker: 55% (krav: 60%) → BREAKING POINT
```

### Treningsplan-justering

Når breaking points er identifisert, justeres treningsplanen:

| Breaking Point | Ekstra fokus | Timer/uke |
|----------------|--------------|-----------|
| Wedge PEI | +2 wedge-økter | +2t |
| Bunker | +1 bunker-økt | +1t |

**Prinsipp**: Maks 30% av treningstid på breaking points for å unngå over-fokus.

## 7. PROGRESJON OG OVERGANGSKRITERIER

### Opprykk til neste kategori

**Krav for kategori-overgang (alle må oppfylles)**:

1. **Score-krav**: 3 måneder konsekvent i mål-kategoriens område
2. **Fysisk**: Minst 2 av 3 fysiske tester bestått
3. **Benchmark**: Minst 4 av 7 golf-tester bestått
4. **Mental**: Trener-godkjenning av modenhet

### Hybrid-kategorier

Ved grensetilfeller (f.eks. score 76.2, mellom D og E):

```
Uke 1-4:   70% E-plan, 30% D-plan
Uke 5-8:   60% E-plan, 40% D-plan
Uke 9-12:  50% E-plan, 50% D-plan
(Evaluér ved benchmark-uke)
```

## 10. SAMMENDRAG

### Nivåprinsippet i praksis:

```
Spiller → Kategori (A-K) → Periodisering (E/G/S/T) → Ukeplan
                ↓
        Breaking Points → Justert fokus
                ↓
        L-fase + CS-nivå + S-setting = Økt-innhold
                ↓
        Benchmark hver 3. uke → Evaluering → Justering
```

### Nøkkelprinsipper:

1. **Kategori-tilpasset**: Alle planer starter fra spillerens kategori
2. **Periodisert**: 4 klare perioder med ulikt fokus
3. **Progressiv**: L1→L5, CS40→CS100, S1→S10
4. **Evidensbasert**: 20 tester måler fremgang
5. **Fleksibel**: Hybrid-kategorier og breaking point-justering
6. **Alderstilpasset**: Juniorspesifikke tilpasninger (multi-sport, lek)
