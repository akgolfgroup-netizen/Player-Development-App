## LANDSCAPE vs PORTRAIT

### Portrait (768x1024)
- Side nav: 240px
- Main content: Single column or 2-column grid
- Modals: Centered, max-width 600px

### Landscape (1024x768)
- Side nav: Collapsed to icons (80px) or full (240px)
- Main content: 2-3 column grid, split views
- Modals: Wider, max-width 800px
- More data visible at once
