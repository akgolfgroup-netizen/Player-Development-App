## 16. Navigasjon per rolle

### Player-navigasjon
```
Dashboard  │  Trening  │  Utvikling  │  Plan  │  Mer
    │           │            │          │        │
    │           ├─ Logg økt  ├─ Stats   ├─ Årsplan├─ Profil
    │           ├─ Dagbok    ├─ Tester  ├─ Kalender├─ Innstill.
    │           ├─ Øvelser   ├─ Fremgang├─ Mål    ├─ Video
    │           └─ Planer    └─ Benchm. └─ Turner.└─ Meldinger
    │
    └─ Quick actions: Logg, Test, Mål
```

### Coach-navigasjon
```
Dashboard  │  Spillere  │  Analyse  │  Plan  │  Mer
    │           │            │          │        │
    │           ├─ Liste     ├─ Stats   ├─ Årsplan├─ Profil
    │           ├─ Grupper   ├─ Sammenl.├─ Kalender├─ Innstill.
    │           ├─ Samlinger ├─ Rapport ├─ Økter  ├─ Øvelser
    │           └─ Varsler   └─ Varsler └─ Saml. └─ Meldinger
    │
    └─ Quick actions: Se varsler, Ny plan, Melding
```

### Admin-navigasjon
```
Oversikt  │  Brukere  │  Konfig  │  Logger  │  Betaling
    │          │          │          │           │
    │          ├─ Spillere├─ Kategori├─ Audit   ├─ Dashboard
    │          ├─ Trenere ├─ Tester  ├─ Feil    ├─ Abonnement
    │          ├─ Pending ├─ Varsler └─ Aktivitet└─ Fakturaer
    │          └─ Inviter └─ Tier
    │
    └─ Quick actions: Godkjenn, Konfig, Rapport
```
